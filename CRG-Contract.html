1. Project & Deliverables

The Developer will design, develop, deploy, and hand over the CRG Dashboard, a custom, mobile-first real-estate operations platform (the "Software"), including the following features:

  • Secure user accounts and login (email + password) with team-gated registration;
  • Role-based access control with six roles and a Super Admin permissions control panel;
  • Transactions / Deals pipeline (CRM) with stages, search, filters, and deal records;
  • Team Directory with photo uploads, contact actions, and role assignment;
  • New-Agent Onboarding workflow with tasks, document uploads, and approvals;
  • Training & Resources module; Announcements; Email-change approval workflow;
  • Real-time in-app notifications; reporting dashboard (pipeline, agent performance, monthly closings);
  • Calendar, video, forms, and social embeds; mobile-first design installable as a PWA;
  • Deployment to Vercel + Supabase; and written handover documentation.

A detailed feature description is provided in the accompanying Quotation (QUO-2026-001), which is incorporated by reference.
